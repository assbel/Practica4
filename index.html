Fecha Evento
1957
Creación de ARPA (Advanced Research Projects Agency).
1969
Creacion de ARPAnet.
1982
Creación del protocolo TCP/IP, que ha sido usado hasta la actualidad.
1983 ARPAnet cambia NCP (Network Control Program) por TCP/IP.
1989 Los protocolos OSI se integran a ARPAnet.
1990
ARPAnet se desintegra, dando lugar a MILnet e Internet.
1991
Tim Berners-Lee crea la World Wide Web, también conocida como WWW, W3 o la Web
2000
Empieza la web 2.0 También conocida como Web Social, el diseño, dinamismo y colaboración de las páginas web incrementa.
2014
HTML5 fue liberado, creando una experiencia agradable para la construcción de aplicaciones web. También es donde se considera el diseño responsivo y el IoT (Internet of Things).
2010 - Ahora

Aparece la Web 3.0, llamada Web Semántica, dado que integra funciones para que se puedan buscar palabras en las páginas web naturalmente.

Aparece la Web 4.0, la Web Ubicua. Integra la inteligencia artificial.
